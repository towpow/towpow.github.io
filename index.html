What is TowPow?

TowPow combines tabletop strategy and an immersive Virtaul reality (VR) fighting experience where a hero is set on the battlefield to battle incoming enemies while the general keeps a strategic overview in order to reach success.

The project is combing the VR headset from HTC Vive and the touch sensitive table called Microsoft Surface. In this project we used the Samsung SUR40 that utililzes the PixelSense technology to handle touch and object input.

Goals

We are striving towards responsive interfaces across devices and have as a goal to have the playing experience to work as seamless as possible on all devices.

Another goal for the TowPow project is to explore asymmetrical gameplay for VR and encourage cooperative gameplay through the HTC Vice and Microsoft's PixelSense.

By integrating physical objects into VR TowPow lets users explore new ways of interacting with a VR enviroment.

Technologies used

TowPow creates the Virtual Reality experience through the HTC Vive headset. We also integrated Microsoft Surface in order to explore multiplayer functionalities in TowPow and to examine programming for a device that can handle fiducial input as well as touch input. The network solution is solved using Unitys High Level Scripting API. All of our own designed 3D models are created using Blender.
